# goproxy

TCP proxy and dumper on go.

## Install

```
go get github.com/yanzay/goproxy
```

## Usage

```
Usage of goproxy:
  -dump string
    	Write dump to file
  -host string
    	Remote host
  -listen string
    	Local address to listen (default ":4242")
  -port int
    	Remote port
  -skip-healthcheck
    	Skip healthcheck
```
